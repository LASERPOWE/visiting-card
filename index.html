Single File Static App

Business Card to Structured Contact Table

Visiting card upload karo, OpenAI se fields extract karvao, aur clean HTML table me saari details dekho. Yeh page GitHub Pages jaisi static hosting par chal sakta hai, kyunki pure frontend me hi request bheji ja rahi hai.

Expected fields

Name, designation, company, mobile, phone, fax, email, website, office address, city, source file, confidence note, aur raw extracted summary. Rotated card images bhi handle karne ke liye prompt tuned hai.

Upload + OpenAI Setup

Agar aap key hardcode karna chahte ho to script me `DEFAULT_OPENAI_KEY` set kar sakte ho.

Multiple card images select ya drag-drop kar sakte ho.

Agar Google Drive image use karni hai to best result ke liye direct image/open link ya publicly accessible file link use karein.

Optional. Agar Google Apps Script webhook doge to uploaded card image ka save request bhej diya jayega.

Optional. Agar Google Apps Script webhook doge to extracted row Google Sheet me save request bhej diya jayega.
